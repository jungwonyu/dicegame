#### 🛠 사용 기술: Html, Css, JavaScript, React
#### 📚 내용 정리:
##### 명령어
- 프로젝트 생성하기: npm init react-app .
- 개발모드 실행하기: npm start (npm run start)
- 실행 중인 서버 종료하기: ctrl + c
- 개발된 프로젝트 빌드하기: npm run build
- 빌드한 것 로컬에서 실행하기: npx serve build
##### 공부내용
- JSX

- State
- React에서의 CSS 적용방법
